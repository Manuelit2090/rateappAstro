P: ¿Por qué change astro.config.mjs?
R: El modo 'static' no soporta API routes dinámicas. 'hybrid' las habilita.

P: ¿Necesito reinstalar node_modules?
R: Sí. npm install agregará bcryptjs y jsonwebtoken.

P: ¿Qué es el flag Secure en cookies?
R: Indica al navegador que solo envíe la cookie por HTTPS (en producción).

P: ¿localStorage es inseguro?
R: Sí, cualquier JavaScript puede accederlo (vulnerable a XSS).

P: ¿Cómo verifico que los cambios funcionan?
R: Sigue las instrucciones POST-CAMBIOS más arriba (PASO 4).

P: ¿Qué debo cambiar en producción?
R: JWT_SECRET, credenciales BD, y habilitar HTTPS. Ver CHECKLIST.
